# Canvas (canvas)

Provides drawing APIs for working with the `canvas` component.

## Methods

- `wx.createCanvasContext(string canvasId, Object this)`: Creates a CanvasContext drawing context object for a canvas.
- `wx.canvasToTempFilePath(Object object, Object this)`: Exports the content of a specified area of the current canvas as an image of the specified size.
- `wx.canvasGetImageData(Object object, Object this)`: Gets the implicit pixel data of a canvas area.
- `wx.canvasPutImageData(Object object, Object this)`: Draws pixel data onto the canvas.
