# System (system)

Provides APIs for retrieving system-related information.

## Methods

- `wx.getSystemInfo(Object object)`: Asynchronously gets system information.
- `wx.getSystemInfoSync()`: Synchronously gets system information.
- `wx.getDeviceInfo()`: Gets basic device information.
- `wx.getWindowInfo()`: Gets window information.
- `wx.getAppBaseInfo()`: Gets basic WeChat app information.
- `wx.getAppAuthorizeSetting()`: Gets WeChat app authorization settings.
