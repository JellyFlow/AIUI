# Calendar

The `calendar` component is used to display date information and supports date browsing, selection, and simple schedule interactions.

## Usage

```xml
<calendar
  current-date="2026-05-02"
  bindchange="handleDateChange"
></calendar>
```

## Features

- Supports displaying content by date.
- Supports date selection and view switching.
- Suitable for scheduling, appointment selection, and time browsing scenarios.
