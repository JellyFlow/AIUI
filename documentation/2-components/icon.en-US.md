# Icon

The `icon` component is used to render icons, usually based on an icon font library such as Material Design Icons. It is essentially a special `text` component.

## Usage

```xml
<icon class="material-icons">home</icon>
```

## Properties

The `icon` component inherits all properties of the `text` component.

| Property | Type | Description | Default |
|-----------|------|-------------|---------|
| `font-size` | Number | The pixel size of the icon. | Inherited |
| `color` | String | The color of the icon. | Inherited |

## Styling

Typically, you can style icons by setting `font-family` and other text-related properties through CSS.
