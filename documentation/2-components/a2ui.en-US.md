# A2UI AI Component

The `a2ui` component is used to host native UI capabilities in AI interaction scenarios. It is suitable for building interfaces for voice interaction, multi-turn conversations, and intelligent task flows.

## Usage

```xml
<a2ui
  agent-id="assistant"
  session-id="{{sessionId}}"
  bindmessage="handleMessage"
></a2ui>
```

## Features

- Provides an interaction container for AI conversations and task execution flows.
- Suitable for hosting message streams, status feedback, and intelligent action entry points.
- Can be combined with business pages to build a native AI application experience.
