# View Containers

View containers are used to organize page structure, hold content, and manage scrolling behavior. They are the foundational capability for building interfaces.

This section mainly includes two core components:

- [`view`](/components/view): A basic container component suitable for layout, grouping, and wrapping content.
- [`scroll-view`](/components/scroll-view): A scrollable container suitable for displaying content when it exceeds the visible area.

## How to Choose

| Scenario | Recommended Component |
| :--- | :--- |
| General layout, content wrapping, and structural organization | `view` |
| A content area that needs vertical or horizontal scrolling | `scroll-view` |

## Recommended Reading Order

1. Start with [`view`](/components/view) to understand the basic container and layout organization approach.
2. Then read [`scroll-view`](/components/scroll-view) to learn how to use scrollable areas and related properties.
