# 视图容器

视图容器用于组织页面结构、承载内容和管理滚动行为，是构建界面的基础能力。

这个章节主要包含两个核心组件：

- [`view`](/components/view)：基础容器组件，适合用于布局、分组和包裹内容。
- [`scroll-view`](/components/scroll-view)：可滚动容器，适合用于内容超出可视区域时的滚动展示。

## 如何选择

| 场景 | 推荐组件 |
| :--- | :--- |
| 普通布局、内容包裹、结构组织 | `view` |
| 需要纵向或横向滚动的内容区域 | `scroll-view` |

## 推荐阅读顺序

1. 先阅读 [`view`](/components/view)，了解基础容器和布局组织方式。
2. 再阅读 [`scroll-view`](/components/scroll-view)，掌握滚动区域的使用方法和相关属性。
