# Swiper 轮播容器

`swiper` 组件用于承载多屏内容，并通过滑动手势或程序控制在不同面板之间切换。

## 使用方法

```xml
<swiper current="0" autoplay="true" interval="3000">
  <view class="slide">第一页</view>
  <view class="slide">第二页</view>
  <view class="slide">第三页</view>
</swiper>
```

## 功能特性

- 支持按页滑动切换内容。
- 支持自动播放和定时切换。
- 适合用于引导页、轮播卡片和多步骤展示场景。
