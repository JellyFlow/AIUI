# Swiper

The `swiper` component is used to host multi-screen content and switch between different panels through swipe gestures or programmatic control.

## Usage

```xml
<swiper current="0" autoplay="true" interval="3000">
  <view class="slide">Page 1</view>
  <view class="slide">Page 2</view>
  <view class="slide">Page 3</view>
</swiper>
```

## Features

- Supports page-based content switching by swiping.
- Supports autoplay and timed switching.
- Suitable for onboarding pages, carousel cards, and multi-step presentation scenarios.
