# 组件

Ink 提供了一组由原生能力驱动的 UI 组件，用于构建应用界面。这些组件以 WXML 形式编写，并通过 Skia 进行原生渲染。

## View Containers

- [View](/components/view): 基础容器组件，用于组织页面结构和布局。
- [ScrollView](/components/scroll-view): 支持横向或纵向滚动的视图容器。
- [Swiper](/components/swiper): 用于承载多屏内容切换的滑动容器。

## Basic Content

- [Text](/components/text): 用于显示纯文本内容。
- [Icon](/components/icon): 基于图标字体或资源的图形内容组件。
- [ErrorState](/components/error-state): 用于展示空状态、异常状态或错误提示。
- [Streamdown](/components/streamdown): 支持流式渲染的 Markdown 内容组件。

## Form Components

- [Button](/components/button): 可点击的按钮组件。
- [Input](/components/input): 单行文本输入组件。
- [Textarea](/components/textarea): 多行文本输入组件。

## Date Components

- [Calendar](/components/calendar): 用于日期展示、选择和日程编排的日历组件。

## Media Components

- [Image](/components/image): 用于显示图片内容。
- [LottieView](/components/lottie-view): 用于渲染 Lottie 动画资源。

## Canvas & Chart

- [Canvas](/components/canvas): 提供 2D 绘图上下文和自定义绘制能力。
- [Chart](/components/chart): 用于渲染数据可视化图表。

## AI Components

- [A2UI](/components/a2ui): 面向 AI 交互场景的原生组件能力封装。
