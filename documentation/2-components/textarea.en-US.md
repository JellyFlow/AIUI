# Textarea

The `textarea` component is used to receive multi-line text, making it suitable for long-form input, remarks, and conversation draft editing scenarios.

## Usage

```xml
<textarea
  value="{{message}}"
  placeholder="Please enter detailed content"
  bindinput="handleInput"
></textarea>
```

## Features

- Supports multi-line text input and editing.
- Suitable for entering and displaying longer content.
- Can be used for feedback forms, comment input, and AI prompt editing scenarios.
