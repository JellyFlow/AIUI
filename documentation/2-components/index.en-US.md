# Components

Ink provides a set of UI components powered by native capabilities for building application interfaces. These components are written in WXML and rendered natively through Skia.

## View Containers

- [View](/components/view): A basic container component for organizing page structure and layout.
- [ScrollView](/components/scroll-view): A view container that supports horizontal or vertical scrolling.
- [Swiper](/components/swiper): A sliding container for switching between multiple screens of content.

## Basic Content

- [Text](/components/text): Used to display plain text content.
- [Icon](/components/icon): A graphical content component based on icon fonts or assets.
- [ErrorState](/components/error-state): Used to display empty states, exceptional states, or error prompts.
- [Streamdown](/components/streamdown): A Markdown content component that supports streaming rendering.

## Form Components

- [Button](/components/button): A clickable button component.
- [Input](/components/input): A single-line text input component.
- [Textarea](/components/textarea): A multi-line text input component.

## Date Components

- [Calendar](/components/calendar): A calendar component for date display, selection, and schedule arrangement.

## Media Components

- [Image](/components/image): Used to display image content.
- [LottieView](/components/lottie-view): Used to render Lottie animation assets.

## Canvas & Chart

- [Canvas](/components/canvas): Provides a 2D drawing context and custom rendering capabilities.
- [Chart](/components/chart): Used to render data visualization charts.

## AI Components

- [A2UI](/components/a2ui): A native component capability wrapper for AI interaction scenarios.
