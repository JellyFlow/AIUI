# Input

The `input` component is used to receive single-line text, making it suitable for search, form filling, and short information entry scenarios.

## Usage

```xml
<input
  value="{{keyword}}"
  placeholder="Please enter search content"
  bindinput="handleInput"
/>
```

## Features

- Supports single-line text input and real-time content updates.
- Supports placeholder text.
- Suitable for search boxes, login forms, and basic information entry scenarios.
