# Global Configuration (`app.json`)

`app.json` is used to configure the global properties of an agent, such as page paths, window styles, and multi-tab settings.

## Agent Description File (`AGENTS.md`)

In addition to `app.json`, each agent project also requires an `AGENTS.md` file to declare the agent's metadata, system instructions, and capability permissions.

For more details about the `AGENTS.md` specification, see [AGENTS.md Specification](/AIUI/framework/config-agents).
