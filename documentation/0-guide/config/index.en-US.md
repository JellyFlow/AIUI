# Configure Agents

In AIUI, you can define the agent's identity, application entry, and page behavior through configuration files.

It is recommended to continue reading in the following order:

- [AGENTS.md](/AIUI/framework/config-agents): First learn how to organize the agent's description, capabilities, and system prompts.
- [Global Configuration](/AIUI/framework/config-global): Then learn how `app.json` defines page entries and global window configuration.
- [Page Configuration](/AIUI/framework/config-page): Finally learn how `page.json` supplements page titles, descriptions, and parameter schemas.

Following this order makes it easier to build a configuration model from the entire agent down to individual pages.
