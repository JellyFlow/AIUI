# 配置智能体

在 AIUI 中，你可以通过配置文件定义智能体的身份、应用入口和页面行为。

建议按下面的顺序继续阅读：

- [AGENTS.md](/AIUI/framework/config-agents)：先了解智能体的描述、能力和系统指令应该如何组织。
- [全局配置](/AIUI/framework/config-global)：再了解 `app.json` 如何定义页面入口和全局窗口配置。
- [页面配置](/AIUI/framework/config-page)：最后了解 `page.json` 如何补充页面标题、描述和参数结构。

按照这个顺序阅读，会更容易建立从智能体整体到单个页面的配置认知。
