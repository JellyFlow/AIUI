# 什么是 AIX？

**AIX**，全称为 **AI eXecutable**，即“AI 可执行包”，是 **AIUI 引入并定义** 的标准软件包格式。它是 AIUI 智能体的专有分发格式，具有以下特点：

## 核心特性

- **基于 OAF 规范**: AIX 包内部遵循 **OAF (Open Agent Format)** 开放智能体规范。
- **自包含**: 包含了智能体的所有代码（WXML/WXSS/JS）、配置（JSON）以及静态资源（图片/音频）。
- **版本追踪**: 每个 AIX 包在打包时都会自动生成一个唯一的 UUID `VERSION` 文件，用于版本校验和热更新。
- **资源优化**: 支持在打包过程中对 PNG/JPEG 图像进行无损/有损压缩，并对 JSON 文件进行混淆和体积压缩。
