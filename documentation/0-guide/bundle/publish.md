# 发布到 Rokid Ai 智能体商店

在完成智能体的开发与打包（生成 `.aix` 文件）后，你可以通过 Rokid AIUI Studio 将其发布到智能体商店，供全球用户下载和使用。

## 1. 登录灵珠平台

访问 [Rokid AIUI Studio](https://aiui.rokid.com/) 并使用你的开发者账号登录。如果你还没有账号，请先完成注册与开发者认证。

## 2. 创建智能体应用

1. 在管理后台选择 **“应用管理”** -> **“创建应用”**。
2. 选择应用类型为 **“AIUI 智能体”**。
3. 填写应用的基本信息，包括：
   - **名称**: 智能体在商店中显示的名称。
   - **图标**: 符合设计规范的方形图标。
   - **描述**: 简洁明了的功能介绍。

## 3. 上传 AIX 软件包

1. 在版本管理页面，点击 **“上传版本”**。
2. 将使用 `aix pack` 命令生成的 `.aix` 文件上传至平台。
3. 平台会自动校验包内的 `VERSION` 文件和 `AGENTS.md` 声明。

## 4. 提交审核

1. 确认版本信息无误后，点击 **“提交审核”**。
2. Rokid 团队将对智能体的性能、交互规范以及安全性进行审核。
3. 审核通过后，你的智能体将正式上架至 Rokid Glasses 智能体商店。

## 5. 版本更新

如果你需要发布新功能或修复 Bug：
1. 在本地修改代码并重新执行 `aix pack`。
2. 在 [Rokid AIUI Studio](https://aiui.rokid.com/) 上传新的 `.aix` 文件。
3. 提交新版本审核。审核通过后，用户设备将根据 `VERSION` 自动触发热更新。
