# 快速开始

欢迎使用 AIUI！AIUI 是一个专为 AI + AR 设计的 AI 原生 GUI（AI-Native UI）本地智能体框架，既强调面向智能体的交互与组织方式，也整合设备侧本地能力，帮助开发者构建真正可落地的智能体应用。为了帮助你更高效地开发智能体，我们提供了完整的文档指南：

## 📖 如何使用本文档

你可以通过顶部的导航栏在不同的功能模块之间切换：

- **快速了解**：如果你是第一次接触 AIUI，建议从这里开始。它包含了智能体的核心概念、目录结构以及如何编写你的第一个智能体。
- **智能体框架**：深入了解 AIUI 的运行机制，包括配置规范（`AGENTS.md`、`app.json`）、[框架结构](/AIUI/framework/structure-app)（应用与页面）以及 WXML/WXSS 的参考文档。
- **组件**：详细列出了 AIUI 支持的所有 UI 组件（如 `view`、`scroll-view`、`image` 等）及其属性和用法。
- **API**：提供了 AIUI 的 JavaScript API 参考，包括框架核心 API、开放服务 API、多媒体 API、网络请求以及与微信小程序兼容的 API。
- **开发者工具**：介绍 AIUI 的开发者工具体系，包括 CLI 与真机调试等能力。
- **更新日志**：了解 AIUI 的最新特性和版本修复。

此外，你可以在页面右上角选择不同的文档版本。
