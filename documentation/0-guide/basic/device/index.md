# 设备

AIUI 提供了连接外设和读取设备传感器的能力，让你的智能体可以直接与硬件交互。

目前设备能力主要分为两类：

- **蓝牙**：发现、连接和读写 BLE 外设。
- **IMU**：读取加速度计、陀螺仪和绝对方向传感器等运动与姿态数据。

## 如何选择

- 想连接心率带、控制器或其他 BLE 外设：使用 **蓝牙**
- 想检测设备移动、晃动或旋转状态：使用 **IMU**
- 想同时使用多种传感器做姿态估计或运动判断：组合使用蓝牙 + IMU
