# AI 能力

AIUI 内置了核心 AI 能力，让你的智能体可以听懂用户、说出来、并完成理解和推理。

这三类能力通常组合使用，形成完整的语音智能体链路：

- **语音识别 (ASR)**：把用户说的话转成文本。
- **大语言模型 (LLM)**：理解上下文并生成回复。
- **语音播报 (TTS)**：把生成结果再读给用户听。

## 如何选择

- 想让应用"听懂"用户输入：使用 **语音识别**
- 想让应用把文本"说出来"：使用 **语音播报**
- 想让应用完成问答、总结、生成、推理：使用 **大语言模型**

## 典型流程

一个完整的语音智能体交互链路通常是这样：

```
用户说话 → 语音识别 → 文本输入 → 大语言模型 → 文本输出 → 语音播报 → 用户听到回复
```
