# CLI

AIUI CLI 是面向开发者的命令行工具入口，用于串联预览、打包与基础信息查看等常见工作流。

## 查看帮助

你可以通过以下命令查看 CLI 的帮助信息：

```bash
aiui --help
```

## 支持功能

- **预览项目**

```bash
aiui open <path>
```

用于打开并预览指定路径下的 AIUI 项目。

- **打包项目**

```bash
aiui pack
```

用于将当前 AIUI 项目打包为可分发产物。

- **查看版本**

```bash
aiui --version
```

用于查看当前 AIUI CLI 的版本信息。
