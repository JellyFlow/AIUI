# CLI

AIUI CLI is the command-line entry point for developers, used to connect common workflows such as previewing, packaging, and viewing basic information.

## View Help

You can view the CLI help information with the following command:

```bash
aiui --help
```

## Supported Features

- **Preview a Project**

```bash
aiui open <path>
```

Used to open and preview the AIUI project under the specified path.

- **Package a Project**

```bash
aiui pack
```

Used to package the current AIUI project into a distributable artifact.

- **View Version**

```bash
aiui --version
```

Used to view the current version information of AIUI CLI.
