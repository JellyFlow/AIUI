# 开发者工具

AIUI 的开发者工具聚焦在四类核心能力上：工具链总览、Craft 工作台、Skills 能力扩展，以及真机调试能力。它们共同构成了从本地开发到页面验证再到设备联调的基础工具链。

## 工具目录

- **介绍**：帮助你理解 AIUI 开发者工具的整体定位与使用场景。
- **Craft 平台**：面向 AIUI 与 Ink 工程的一体化工作台，用于导入项目、编辑文件和实时预览页面。
- **Skills**：提供面向 AIUI 开发的专用上下文与规范，帮助你在智能体开发场景中获得更准确的生成与协作结果。
- **真机调试**：帮助你在真实设备环境中排查问题、验证交互与观察运行状态。
