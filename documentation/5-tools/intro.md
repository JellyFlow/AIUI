# 开发者工具 (Developer Tools)

AIUI 的开发者工具聚焦在四类核心能力上：工具链总览、Craft 工作台、命令行工作流，以及真机调试能力。它们共同构成了从本地开发到页面验证再到设备联调的基础工具链。

## 工具目录

- **介绍**：帮助你理解 AIUI 开发者工具的整体定位与使用场景。
- **CLI**：面向项目创建、构建、运行与调试的命令行入口。
- **Craft 平台**：面向 AIUI 与 Ink 工程的一体化工作台，用于导入项目、编辑文件和实时预览页面。
- **真机调试**：帮助你在真实设备环境中排查问题、验证交互与观察运行状态。
