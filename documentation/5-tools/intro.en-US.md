# Developer Tools

AIUI developer tools focus on four core capabilities: a tooling overview, the Craft workspace, Skills-based capability extension, and on-device debugging. Together, they form the foundational toolchain from local development to page verification and device integration.

## Tool Directory

- **Introduction**: Helps you understand the overall positioning and usage scenarios of AIUI developer tools.
- **Craft Platform**: An integrated workspace for AIUI and Ink projects, used to import projects, edit files, and preview pages in real time.
- **Skills**: Provide AIUI-focused context and conventions so agents can generate code and collaborate more accurately in development workflows.
- **On-Device Debugging**: Helps you troubleshoot issues, verify interactions, and observe runtime status on real devices.

Please select a specific section from the left-side menu to view detailed instructions.
